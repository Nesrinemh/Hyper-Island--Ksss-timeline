useContentful() → {Object}

useContentful is a hook that creates a Contentful client and provides a function to fetch and sanitize data from Contentful. The client is configured with a space ID, an access token, and a host. The getData function fetches data of the 'ksssEvent' content type, selects the 'fields' of each entry, and orders the entries by 'fullDate'. Each entry is sanitized into a new object that includes the entry's fields, images, and ID.
